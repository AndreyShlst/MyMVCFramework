Это пример простого фреймворка с MVC-архитектурой,создан ради научного интереса и фана

Структура: application - содержит файлы фреймворка (Контроллеры,Модели,Шаблоны) data - содержит файл с информацией о зарегистрированых пользователях Что он уже умеет: Здороваться=) - user/hello/name/"Имя" (В адресной строке) Показать список всех пользователей(+права) - user/list Показать права заданого пользователя - user/get/role/"Имя" Добавить нового пользователя - user/add/name/"Имя"/role/"Права" Храни Вас эстетика!
